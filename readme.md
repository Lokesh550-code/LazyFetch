LazyFetch

A lightweight Neofetch-inspired system info tool built with Node.js.
Designed for people who enjoy tinkering with their terminal and want a customizable cross-platform alternative.

=====================================================================================================================

Features

Cross-platform (works on Windows, Linux, macOS)

Built using Node.js

Fetches system information directly from the OS

Simple developer-friendly setup

Planned full CLI support

========================================================================================================================
Tech Stack

Node.js

JavaScript

npm scripts

=========================================================================================================================
Installation

Clone the repository: git clone https://github.com/Lokesh550-code/LazyFetch.git


Move into the project directory: cd LazyFetch


Install dependencies:

npm install

Run the Project

Currently runs in development mode:

npm run dev
